# Tripleten Proyecto 4: web_project_homeland

La página web de la Librería Triple Peaks es el primer proyecto en el programa de Ingeniería de software en Triple Ten. Fue creado utilizando HTML y CSS, con base en un informe de diseño conforme a distintos tamaños de resolucion (1280px, 768px, 320px) mediante el uso de media querys.

## Características del proyecto

-	HTML5 semántico
-	Grid
-	Posicionamiento
-	Git-hub
-	Apilamiento vertical con z-index
-	Diseño responsivo
-	Media Querys
